## **开始使用**

要在您的 CrewAI 项目中使用任何工具（Tool）：

1. **导入（Import）** 工具到您的团队（Crew）配置中
2. **添加（Add）** 到您的智能体（Agent）工具列表中
3. **配置（Configure）** 任何必需的 API 密钥或设置

```python
from crewai_tools import FileReadTool, SerperDevTool

# 向您的智能体添加工具
agent = Agent(
    role="研究分析师",
    tools=[FileReadTool(), SerperDevTool()],
    # ... 其他配置
)
```
